ℹ️ About PureWolf

PureWolf Native is required for this extension to connect to your computer using Firefox Native Messaging.

It allows PureWolf to:

  • Start and control the local TELA server
  • Read your TELA folders
  • Serve TELA sites as real web pages
  • Search and resolve SCIDs via the indexer

🛠 Installation

  1. Open your Terminal and Run: 
    curl -fsSL https://github.com/ArcaneSphere/PureWolf-Browser-Extension/releases/download/v1.0.0/install.sh | bash
  2. Restart Firefox after installation.

Visit the GitHub releases page for more information on the PureWolf Native easy one-command package installer.

🔗 Connect

  • Use your own local node for best performance and reliability.
  • Public nodes also work well — find them on the Bookmarks page.
  • Public nodes: first Gnomon sync ~17 seconds, SCID loading 3–15 seconds or more depending on the application size.
  • Local nodes: sync quickly, SCIDs load nearly instantly.
